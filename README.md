### 게임 규칙
+ 두 사람이 주사위를 세 개씩 굴림
+ 중복되는 숫자가 없을 시 세 숫자의 합이 높은 경우 승리
+ 중복되는 숫자가 있는 경우 중복되는 숫자를 가진 주사위가 많은 쪽이 승리
+ 중복되는 숫자를 가진 주사위의 개수가 같은 경우 중복된 숫자가 높은 쪽이 승리
